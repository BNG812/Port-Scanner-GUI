# 🛡️ Port & Service Scanner (GUI) - Python

## 📌 Project Overview
This is a Python-based GUI Port Scanner that scans a target system to identify open ports and running services.

The tool is designed for learning purposes to understand how network reconnaissance works in cybersecurity.

---

## 🚀 Features
- GUI-based interface (Tkinter)
- Multi-threaded port scanning
- Service detection (HTTP, FTP, SSH, etc.)
- Custom port range scanning
- Real-time scan results display
- Terminal-style output

---

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- Socket Programming
- Threading

---

## ⚙️ How It Works
1. Enter target (IP/Domain)
2. Select port range
3. Click "Execute Scan"
4. Tool scans ports and displays open ones
